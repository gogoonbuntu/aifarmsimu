# 🍓 딸기 (Strawberry, *Fragaria × ananassa*)

![딸기](https://images.unsplash.com/photo-1464965911861-746a04b4bca6?w=640&q=80)
*딸기 수확시기*

## 분류
- **과**: 장미과 · **카테고리**: 과수 (시설 재배) · **품종**: 설향, 금실, 죽향, 비타베리

## 🏆 지역별 유명 산지

| 지역 | 특징 |
|------|------|
| **논산** (충남) | 한국 딸기 수도. 전국 생산량 1위, 설향 품종 발원지 |
| **담양** (전남) | 죽향 특산지, 대나무 유기질 퇴비 활용 |
| **진주** (경남) | 남부 시설재배 확대, 11월~이듬해 5월 연속 출하 |
| **산청** (경남) | 고냉지 여름딸기 (6~10월) 프리미엄 시장 |
| **양평** (경기) | 체험 농장 위주, 수도권 직거래 |

### 📋 실제 농사 사례
> **논산 설향 사례** (2023, 충남농업기술원)  
> 단동 비닐하우스 700평. 9월 10일 정식, 12월~5월 수확.  
> 야간 최저 8°C 유지 (보온커튼 + 온풍기).  
> 수량 **3,800 kg/10a**, Brix 12.5, S등급.  
> 핵심: 12시간 이하 단일조건에서 화아분화 유도 후 정식.

> **산청 여름딸기** (2022)  
> 해발 500m 고냉지. 야간 15°C → 여름에도 착과.  
> 8월 Brix 10.8. kg당 **15,000원** (비수기 프리미엄).

## 환경 요구
- 최적: 18/8°C · 치사 저온 -10°C · 고온 35°C (고온 매우 약함)
- **단일성**: 일장 12h 이하에서 화아분화
- **NPK**: 8:5:10 (K 높음)
- 병해: 흰가루병(4%), 잿빛곰팡이병(5%)
