# 🧅 대파 (Green Onion, *Allium fistulosum* L.)

![대파](https://upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Welsh_onion_flowers.jpg/320px-Welsh_onion_flowers.jpg)
*대파 꽃 (Wikimedia)*

## 분류
- **과**: 백합과 (Liliaceae) · **카테고리**: 채소 · **원산지**: 시베리아~중국 서부

## 생육 모델
| 단계 | GDD | 기간 |
|------|-----|------|
| 발아기 | 60°C·일 | 7~14일 |
| 유묘기 | 200°C·일 | 20~30일 |
| 엽초생장기 | 1,200°C·일 | 60~100일 |
| 비대기 | 400°C·일 | 30~50일 |
| 수확적기 | — | 14~30일 |

- **기본온도**: 4°C · **총 GDD**: 2,100°C·일 · **RUE**: 1.8 · **HI**: 0.65
- 평균 수량: 4,000 kg/10a

## 환경 요구
- 최적: 18/12°C · 치사 저온: -5°C · 내한성 강 · 치사 고온: 38°C
- 수분: 500~700mm · **NPK**: 12:4:6 (N 중심)
- 적합 토양: 사양토, 충적양토 · pH 6.0~7.0
