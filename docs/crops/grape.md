# 🍇 포도 (Grape, *Vitis vinifera* L.)

![포도](https://images.unsplash.com/photo-1596363505729-4190a9506133?w=640&q=80)
*포도 착색성숙기*

## 분류
- **과**: 포도과 (Vitaceae) · **카테고리**: 과수 (다년생, C₃)
- **국내 품종**: 캠벨얼리, 거봉, MBA, **샤인머스캣** (최근 주력)

## 🏆 지역별 유명 산지

| 지역 | 특징 |
|------|------|
| **영동** (충북) | 한국 포도 수도. 내륙 분지 일교차 12°C → 당도 극대화 |
| **김천** (경북) | 샤인머스캣 전국 1위 생산량 (2023) |
| **상주** (경북) | 거봉 포도 특구, 사양토 배수 우수 |
| **천안** (충남) | 거봉 특산지, 수도권 출하 유리 |
| **안성** (경기) | 캠벨얼리 전통 산지 |

### 📋 실제 농사 사례
> **영동 샤인머스캣 사례** (2023, 충북농업기술원)  
> 연동 비닐하우스 1,500평. 3월 발아 → 9월 20일 수확.  
> 비가림 재배로 잿빛곰팡이병 90% 억제.  
> Brix **19.2**, 과립중 12.8g. kg당 **12,000원**.  
> 핵심: 과실비대기 **일교차 10°C+** 유지 (야간 측창 개방).

> **김천 거봉 사례** (2022)  
> 노지 비가림 재배. 7월 장마기 흰가루병 발생 → 유황제 방제.  
> 최종 Brix 17.5, B등급. 수량 1,400 kg/10a.

기본온도 10°C · 총 GDD 1,800°C·일 · RUE 1.5 · HI 0.40

## 환경 요구
- 최적: 25/15°C · 치사 저온 -20°C · 치사 고온 42°C
- **일교차 10°C+**: 당 축적 15~25% 증가 (Kliewer, 1977)
- 수분: 600~800mm · **NPK**: 5:3:8 (K 높음)
- 적합 토양: 사양토 · pH 5.5~7.0 · 배수 매우 양호
