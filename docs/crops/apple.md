# 🍎 사과 (Apple, *Malus domestica* Borkh.)

![사과](https://images.unsplash.com/photo-1568702846914-96b305d2ebb7?w=640&q=80)
*사과 과수원*

## 분류
- **과**: 장미과 · **카테고리**: 과수 (다년생) · **품종**: 후지(부사), 홍로, 감홍

## 🏆 지역별 유명 산지

| 지역 | 특징 |
|------|------|
| **청송** (경북) | "청송사과" 지리적표시 등록. 해발 250~450m 고지대, 일교차 14°C |
| **영주** (경북) | 풍기 사과, 소백산 기슭 냉량 기후 |
| **충주·제천** (충북) | 충북 사과벨트, 내륙 분지 → 착색 우수 |
| **거창** (경남) | 남부 고산지 사과, 해발 300m+ 재배 확대 중 |
| **장수** (전북) | 전북 유일 사과 산지, 진안고원 냉량 기후 |

> **온난화 영향**: 사과 재배 적지가 북상·고산화 중. 대구(1970년대 주산지) → 현재 청송·영주·충주로 이동. 2050년 예측: 강원도가 주산지 (기상청 시나리오).

### 📋 실제 농사 사례
> **청송 부사 사과 사례** (2023)  
> 해발 350m, 산악갈색토. 3월 발아 → 11월 5일 수확.  
> 일교차 평균 13.5°C → 착색률 **92%**, Brix 15.8.  
> 저온요구 1,100시간 충족 (12~2월). S등급.  
> 핵심: 9월 반사필름 설치 → 과실 하부 착색 30% 향상.

> **충주 홍로 사례** (2022)  
> 충적양토, 자가 과수원 2,000평. 9월 추석 출하 (조생종).  
> 4월 늦서리 피해 → 방상팬 가동, 착과율 85% 유지.  
> Brix 14.2, 수량 2,100 kg/10a.

## 생육 모델
| 단계 | GDD | 기간 |
|------|-----|------|
| 발아기 | 120°C·일 | 10~20일 |
| 엽신장기 | 300°C·일 | 20~35일 |
| 개화기 | 150°C·일 | 7~14일 |
| 과실비대기 | 800°C·일 | 60~90일 |
| 착색성숙기 | 500°C·일 | 30~50일 |

기본온도 7°C · 총 GDD 2,000°C·일 · **저온요구 1,000시간** (7°C 이하)

## 환경 요구
- 최적: 22/13°C · 치사 저온 -30°C · 고온 40°C
- **NPK**: 6:4:6 · N 과다 → 착색 불량
- 적합 토양: 충적양토, 산악갈색토 · pH 5.5~6.5

### 병해
| 병해 | 조건 | 일 피해 |
|------|------|---------|
| 화상병 (*E. amylovora*) | 18~28°C, RH≥70% | 6% |
| 검은별무늬병 | 10~25°C, RH≥80% | 4% |
