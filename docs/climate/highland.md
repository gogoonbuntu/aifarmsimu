# 🏔️ 고냉지 (Highland) — Dfb

![highland](../images/highland.png)

## 기후 개요
연평균 8.5°C · 강수 1,500mm · 무상일수 150일 · 해발 600~1,000m  
대표: 대관령, 평창, 태백

## 🏆 지역 유명 농산물
| 지역 | 특산물 | 근거 |
|------|--------|------|
| **대관령** | 감자, 배추 | 여름 야간 15°C → 괴경 비대·결구 최적 |
| **평창** | 씨감자 | 진딧물 밀도 낮아 바이러스 프리 |
| **태백** | 시금치, 여름배추 | 고냉지 채소 비수기 출하 |
| **정선** | 황기, 약초 | 산지 약용작물 |

## 특성
- **냉량 기후**: 여름 평균 20°C → 감자·배추 최적
- **긴 겨울**: 무상일수 150일, 10월 초 서리
- **큰 일교차**: 야간 저온 → 당 축적

## 추천 작물
감자(4~5월), 배추(6~7월), 사과(3월)
