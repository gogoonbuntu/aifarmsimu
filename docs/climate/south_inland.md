# 🌤️ 남부내륙 (South Inland) — Cwa

![남부내륙](../images/south_inland.png)

## 기후 분류
- **쾨펜 분류**: **Cwa** (온대 동계건조 고온형)
- **연평균 기온**: 13.5°C · **강수**: 1,200mm · **무상일수**: 205일
- **대표 지역**: 대구, 안동, 전주, 광주, 논산

## 기상 특성 ([KMA](https://data.kma.go.kr))
- **대구 분지효과**: 폭염일수 전국 최다 (연 18~25일). 2018년 40.0°C ([역대 최고](https://data.kma.go.kr))
- **큰 일교차**: 내륙 분지 → 주간 고온·야간 냉각 → 과수 착색 유리
- **강수 집중**: 7~8월 연 강수의 55~60%
- **태풍**: 연 1.5회 직접 영향 (9월 과수 낙과 주의)
