# 🏖️ 사양토 (Sandy Loam) — Inceptisol

![사양토](../images/sandy_loam.png)

## USDA 분류: [Inceptisol](https://www.nrcs.usda.gov/resources/guides-and-instructions/soil-taxonomy)
모래 함량 높아 배수 매우 양호. 뿌리채소·과수에 최적.

## 물리·화학적 특성
| 항목 | 값 |
|------|------|
| 토성 | 사양토 (Sand 65% · Silt 25% · Clay 10%) |
| pH | 5.0~6.0 |
| 유기물 | 2.0% |
| 포장용수량 | 0.22 · 위조점 0.08 |
| 유효수분 | **140 mm/m** (낮음) |
| CEC | 11 cmol⁺/kg |
| 유효토심 | 70cm |
| 배수 | **매우 양호** (Ksat 30 mm/day) |

## 작물 적합도
| 작물군 | 적합도 | 이유 |
|--------|--------|------|
| 근채 | ★★★★★ (0.90) | **감자·고구마 최적** — 수확 용이, 괴경 형태 양호 |
| 과수 | ★★★★☆ (0.85) | 포도에 특히 좋음 (당도 집중) |
| 채소 | ★★★★☆ (0.70) | 가능, 관수 관리 필수 |
| 벼 | ★★☆☆☆ (0.30) | **불적합** — 담수 유지 곤란 |

> ⚠️ **보수력 낮음**: 유효수분 140mm/m — 가뭄 시기 관수 관리 필수. 점적관수 추천.

## 분포
낙동강 유역, 해안가, 경기 북부, 여주. 양분: N 80 · P 60 · K 100 mg/kg
